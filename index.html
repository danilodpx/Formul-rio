Responda as perguntas abaixo:

  1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?

    2. Especifica uma dica curta que descreve o valor esperado de uma área de texto.

  2. Na tag input com atributo type="radio" ou type="radio", para que serve o atributo "checked"?

    3. Checked é uma caixa de marcação. Você deve usar o atributo value para definir o valor enviado por este item. Use também o atributo checked para indicar se o item está selecionado por padrão.

  3. Na tag input para que serve o atributo "value"?

    4. O atributo value serve para definir o valor do item. O value atributo é usado de forma diferente para diferentes tipos de entrada:
    Para "botão", "redefinir" e "enviar" - define o texto no botão.
    Para "texto", "senha" e "oculto" - define o valor inicial (padrão) do campo de entrada.
    Para "checkbox", "radio", "image" - define o valor associado à entrada (este também é o valor que é enviado no envio).

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    5. type="radio"

  5. Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

    6. A tag "label" representa uma legenda para um item em uma interface de usuário. Já a tag "for" faz um tipo de link entre a label e o input . Logo o for tem que apontar para o ID do input.

  6. Na tag "option" para que serve o atributo "selected"?

    7. Especifica que uma opção deve ser pré-selecionada quando a página for carregada.

  7. Nas tags de formulário para que serve o atributo "required"?

    8. Esse é um atributo booleano usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.

  8. Na tag "form" para que serve o atributo "action"?

    9. Especifica para onde enviar os dados do formulário quando um formulário é enviado.

  9. Na tag "form" para que serve o atributo "method"?

    10. Especifica o método HTTP a ser usado ao enviar dados de formulário.

  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?

    11. "post"

  11. Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?

    12. "get"
